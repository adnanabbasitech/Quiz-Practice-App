 # 📘 Quiz Practice App

A responsive and interactive **Quiz Practice Web Application** built using **HTML, CSS, JavaScript, and Bootstrap 5**.  
It allows users to attempt multiple-choice questions from **General Knowledge and Science** with a timer-based system and instant result evaluation.

## ✨ Features

- 🧠 Multiple-choice quiz system  
- ⏱️ 30-second timer per question  
- 🔄 Next / Previous navigation  
- 📊 Live score tracking (Correct / Attempted)  
- 🏁 Final result screen showing:
  - Total Score  
  - Correct Answers  
  - Attempted Questions  
  - Total Time Taken (minutes + digital format)  
- 🎨 Responsive UI using Bootstrap 5  
- 📱 Mobile-friendly design
